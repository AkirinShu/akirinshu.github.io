赣鄱幻想大结界

一、基本信息

杂志名称： 赣鄱幻想大结界
发起与运营： 梦之朝霭社团/江西东方Project同好会/空想豫章筹办组
发起时间： 2024年春~2025年11月
杂志定位： 以江西省为范围的非营利性东方Project同人杂志；作为江西东方的信息聚合地 / 作品发布平台 / 社群记录者；线上以赣T官网文章、B站专栏、小红书图文、微信公众号等形式发布；线下以实体杂志形式在同人展（如赣T）、聚会、社群内部发行；强调文化传承、同人创作激励、社区凝聚与展示平台的构建。
发行周期： 季刊（春分、夏至、秋分、冬至发布）
受众范围： 以江西省东方同好为核心，全国同好次之，有条件则考虑国际传播。

二、杂志目标与核心价值

1.打通信息差

在社群的海量信息中提取有价值的内容，将其精炼总结，节省同好们获取信息的精力，并让视野从QQ群及各个商业平台构造的信息茧房中扩展出来。

2. 解决同好创作动力不足

许多同好虽有创作倾向，但缺乏动力与目标，且线上的作品往往缺乏面对面社群的仪式感。杂志将提供明确的投稿渠道，通过实体刊物展示带来成就感，提供“目标导向型创作”的机会（如主题征稿），并建立良好的同人文化生态促进作者更愿意更有动力进行创作。

3. 解决作品曝光度不足

作品在群聊中极易被新消息覆盖，且受限于商业平台的流量算法与曝光控制，导致信息茧房和对创作内容自由度的压力。本杂志提供组织化、可长期保存归档的平台，通过多平台（官网、B站等）曝光以及赣T展会的实体售卖，确保持续且稳定的展示空间。

4. 促进代际传承

针对新人不知历史、老人看不见新人的断层现象，我们需要对江西社群多年的内容进行存档。杂志将鼓励新人在刊物中露面，同时引导老人记录江西东方社群的历史。

5. 建立同人文化生态

明确同人文化共识，反对AI生成内容、抄袭及侵权行为。提倡尊重、互评文化与建设性交流，打通画师、作家/文手、Coser、摄影师及印刷厂等人才与资源的互通，鼓励跨社团合作与经验分享。

6. 线上与线下的角色

  • 线上：承担信息聚合，归档社群作品；作为跨平台的去商业化聚合器；为江西同人展提供线上展示空间。
  • 线下：出现在展会现场作为文化载体；保存与展示创作成果；作为江西东方同人文化的纸质档案。

三、杂志栏目规划

为了保持杂志的稳定结构与内容多样性，每期将参考以下栏目：

  • 卷首语：一般由主编或当期嘉宾撰写，概述本期主题或社群近况。
  • 特集：与当季主题相关的重点内容，如角色专题、文化分析、活动历史专题或访谈。
  • 同人作品发表区：包含文字作品（小说、随笔、评论等）与图像作品（插画、漫画、摄影、Cosplay等）。
  • 社群板块：包含投稿留言、问卷调查、各地同好自述、本地创作者/Coser/社团访谈，以及展会活动报道与预告等。
  • 新闻/宣发/广告：包含江西至全国、世界的有价值动态及东方社团广告/作品/作者宣传等。
  • 创作教程/资料板块：分享绘画与写作技巧、工具推荐及东方资料科普等。
  • 公告与投稿须知：发布下期主题、投稿方式及宣传合作说明等。

四、投稿、作品和授权

投稿范围：接受所有东方Project相关且适合纸媒刊登的内容，包括：

  • • 文字作品（短篇/中篇/随笔/诗歌等）；研究/杂文/评论/纪实等；可含插图。
  • • 图像作品（插画/漫画/摄影作品/cosplay照片等）；可含文字说明。
  • • 社团广告/作品/作者宣传。
  • • 其他可印刷的东方相关内容。

    • 格式要求：文字类接受任何普通人可识别出文字的文件（甚至木刻文字的照片）；图片类接受任何非动态图像文件（除非有什么魔法让动图在纸上动）。

    禁止内容：

    • • 带有疑似卖弱假捧、阴阳怪气、火药味的内容；钓鱼，反串的内容；违法违规、敏感的内容；NSFW内容
    • • 未经授权的转载、翻译或二改。
    • • 包含任何AI生成内容的作品。
    • • 其他不适宜内容。

    投稿方式：联系当前杂志主编：秋霖澍（QQ：3541347237 Email：malin233@qq.com）或运营方

    稿费相关：目前杂志运营方无力承担稿费，由同好们自愿投稿。稿件被最终采用的作者们在实体刊物制作完成后都能免费拿到一本。

    授权与版权：作者保留作品全部权利。杂志仅拥有当期刊物的使用权及宣传时的部分展示权。所有授权均为一次性、非独占、非商业性质。如作品适合单独收录进赣T官网，杂志运营方将联系作者取得授权。

    五、传播与发行

    线上传播：发布新刊预告与宣传图，通过B站专栏、小红书、微信公众号等发布部分内容，并向作者发送电子样刊。正式发布时，文章将归档至赣T官网。

    线下发行：在赣T及其他江西东方活动设置摊位。向参与作者提供实体样刊（可邮寄），并邀请作者坐摊进行签售与互动。

    六、组织结构与分工

    为保持非营利性质与可持续运作，采用“志愿者 + 明确分工”模式：

    • 主编/编辑：负责整体方向、栏目规划、质量把控、审稿、沟通及稿费（如果有）安排。
    • 美工/投产：负责整体设计、排版、输出印刷稿件及联系印刷厂。
    • 物流与售卖：由梦之朝霭或空想豫章筹办组货物管理人员负责。

    全员将共同参与校对、栏目规划及宣传运营。随着规模扩大，可增设翻译及专门职位。

    七、经费预算与未来展望

    非营利原则与成本

    杂志采取非营利性运作，条件适合将公开账目。所有收入用于下一期制作。实体刊物售价仅基本覆盖制作成本。

    未来展望

    • • 建立稳定投稿体系与固定读者群。
    • • 成为江西东方同好的中心发布平台。
    • • 完成江西东方同人文化初步档案体系。
    • • 形成成熟的社群参与机制。
    • • 开展线上活动，如创作比赛、主题季、专题号。
    • • 建立区域性的“东方文化数字档案库”。
    • • 成为辐射至全国的公共同人文化平台。
    • • 通过多语言版本向国际传播。
